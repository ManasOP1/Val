# Background Music

- **Recommended:** Download a free track from [Mixkit](https://mixkit.co/free-stock-music/mood/romantic/) or [Pixabay](https://pixabay.com/music/search/romantic/)
- **Filename:** Save as `romantic.mp3` (or `romantic.ogg` for better browser support)
- **Format:** MP3 or OGG
